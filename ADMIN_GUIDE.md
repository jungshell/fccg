| 등급 | 이모지 | 한글명 | 설명 |

✅ **콘텐츠 관리**

## 🗳️ 투표 시스템 규칙

### 투표 세션 생성

**권한**: 👑 슈퍼관리자, 🔵 관리자만 가능

**규칙**:
1. **주간 단위**로 투표 세션 생성
   - 매주 월요일 00:01부터 목요일 17:00까지 투표 진행
   - 같은 주간에 중복 세션 생성 불가

2. **투표 기간**
   - 시작: 월요일 00:01
   - 종료: 목요일 17:00
   - 마감 후 수정 불가

3. **차단 요일 설정**
   - 특정 요일을 투표에서 제외 가능
   - 사유 입력 필수

### 투표 참여

**권한**: 모든 회원 (⚪ MEMBER 이상)

**규칙**:
1. **1인 1투표**
   - 한 세션당 1회만 투표 가능
   - 재투표 시 기존 투표 자동 삭제 후 새 투표 생성

2. **다중 선택 가능**
   - 여러 요일 선택 가능
   - 최소 1개 이상 선택 필수

3. **투표 수정**
   - 마감 전까지 수정 가능
   - 마감 후 수정 불가

### 투표 결과 처리

**권한**: 👑 슈퍼관리자, 🔵 관리자만 가능

**절차**:
1. 투표 마감 후 결과 확인
2. "결과 저장" 버튼 클릭
3. 자동으로 경기 일정 생성
   - 가장 많이 선택된 요일 기준
   - 자동생성 일정은 `autoGenerated: true`로 표시

## ⚽ 경기 관리 규칙

### 경기 생성

**권한**: 👑 슈퍼관리자, 🔵 관리자만 가능

**필수 정보**:
- 📅 경기 날짜
- ⏰ 경기 시간
- 📍 경기 장소
- 🏷️ 행사 유형 (매치/자체/회식/기타)
- 👥 참여자 선택

**특징**:
- 자동생성 일정: 투표 결과로 자동 생성된 경기
- 수동 생성 일정: 관리자가 직접 생성한 경기

### 경기 확정

**권한**: 👑 슈퍼관리자, 🔵 관리자만 가능

**절차**:
1. 경기 일정 생성
2. 참여자 확인
3. "경기 확정" 버튼 클릭
4. 확정된 경기는 `confirmed: true`로 표시

### 경기 참여 관리

**권한**: 모든 회원 (본인 참여 정보만)

**기능**:
- 참여/불참 선택
- 참여 상태 변경 가능

## 👤 회원 관리 규칙

### 회원 등급 변경

**권한**: 
- 👑 슈퍼관리자: 모든 등급 변경 가능
- 🔵 관리자: MEMBER → ADMIN만 가능

**주의사항**:
- ⚠️ SUPER_ADMIN 등급은 슈퍼관리자만 변경 가능
- ⚠️ 본인 등급 변경 불가 (다른 관리자 필요)

### 회원 상태 관리

**상태 종류**:
- ✅ **ACTIVE** (활성): 정상 활동 중인 회원
- ⏸️ **INACTIVE** (비활성): 일시적으로 활동 중단
- 🚫 **SUSPENDED** (정지): 규칙 위반으로 인한 정지
- ❌ **DELETED** (삭제): 탈퇴 또는 삭제된 회원

**자동 정지 규칙**:
- 투표 참여율이 일정 기준 이하인 경우 자동 정지
- 관리자가 수동으로 정지 해제 가능

### 비밀번호 초기화

**권한**: 👑 슈퍼관리자, 🔵 관리자만 가능

**절차**:
1. 회원 정보 수정 모달 열기
2. "비밀번호 초기화" 버튼 클릭
3. 임시 비밀번호 생성 및 전달

## 📊 활동 점수 시스템

### 점수 계산 방식

**경기 참여**: 50점/회
- 경기에 참여한 경우 50점 획득
- 불참 시 점수 없음

**투표 참여**: 30점/회
- 투표 세션에 참여한 경우 30점 획득
- 미참여 시 점수 없음

### 참여율 계산

**경기 참여율**:
```
경기 참여율 = (참여한 경기 수 / 가입 후 총 경기 수) × 100
```

**투표 참여율**:
```
투표 참여율 = (참여한 투표 수 / 가입 후 총 투표 수) × 100
```

**특징**:
- 회원 가입일 기준으로 계산
- 가입 전 경기/투표는 제외

### 활동 점수 활용

- 활동 분석 대시보드에서 확인 가능
- 상위 참여자 랭킹 표시
- 회원 활동 평가 기준

## 🔔 알림 시스템

### 알림 종류

1. **경기 알림** (GAME_REMINDER)
   - 경기 전날 알림
   - 경기 당일 알림

2. **투표 알림** (VOTE_REMINDER)
   - 투표 시작 알림
   - 투표 마감 임박 알림

3. **신규 회원 알림** (NEW_MEMBER)
   - 새 회원 가입 시 관리자에게 알림

4. **회원 정지 알림** (MEMBER_SUSPENDED)
   - 회원 정지 시 알림

### 알림 설정

**권한**: 👑 슈퍼관리자만 가능

**설정 항목**:
- 알림 활성화/비활성화
- 알림 대상 선택 (전체/관리자만)
- 알림 시간 설정

## ❓ 자주 묻는 질문

### Q1. 회원 등급을 어떻게 변경하나요?

**A**: 관리자 페이지 → 회원 관리 → 수정 버튼 클릭 → 등급 선택 → 저장

### Q2. 투표 세션은 언제 생성하나요?

**A**: 매주 월요일 이전에 다음 주 투표 세션을 생성하세요. 자동 생성 기능도 있습니다.

### Q3. 자동생성된 경기 일정을 수정할 수 있나요?

**A**: 네, 관리자는 자동생성된 경기도 수정/삭제할 수 있습니다.

### Q4. 회원이 투표에 참여하지 않으면 어떻게 되나요?

**A**: 일정 기준 이하의 참여율이면 자동으로 정지 상태가 됩니다. 관리자가 수동으로 해제할 수 있습니다.

### Q5. 활동 점수는 언제 업데이트되나요?

**A**: 경기 참여 또는 투표 참여 시 실시간으로 업데이트됩니다.

### Q6. 슈퍼관리자와 관리자의 차이는?

**A**: 슈퍼관리자는 모든 기능에 접근 가능하며, 활동 분석 및 알림 설정 관리가 가능합니다. 관리자는 대부분의 기능을 사용할 수 있지만 일부 시스템 설정은 제한됩니다.

### Q7. 회원 삭제와 정지의 차이는?

**A**: 
- **정지 (SUSPENDED)**: 일시적으로 활동 제한, 복구 가능
- **삭제 (DELETED)**: 영구적으로 제거, 복구 불가
